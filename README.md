# Markdown Reader

Small static browser app for opening and reading Markdown files locally.

## Use

1. Open `index.html` in your browser.
2. Click **Open Markdown File** or drag a `.md` file into the page.
3. Read the rendered preview and edit the source if needed.
4. After you open a real file, a left rail appears with recent file history.
5. If you want to browse sibling files from the same folder, use **Open Folder**.
6. Use **Hide Source** in the preview header to let the preview fill the full viewer width.
7. Use the hamburger button to collapse or reopen the left rail.
8. Use each rail section's own control to collapse or expand `History` and `Same Folder`.

## Notes

- The app uses a dark reading layout.
- Folder browsing depends on browser support for folder selection.
- No build step required.
